Pattern Creator:
	- PrenotazioneCreator utilizza il pattern sulla classe Prenotazione.
	- ClienteCreator utilizza il pattern sulla classe Cliente.
	- Prenotazione utilizza il pattern sulla classe Viaggiatore.
	- Catalogo utilizza il pattern sulla classe PacchettoVacanza.
	- Pagamento utilizza il pattern sulla classe Rimborso.

	- CatalogoController utilizza il pattern sulla classe Catalogo.
	- ClienteController utilizza il pattern sulla classe ClienteCreator.
